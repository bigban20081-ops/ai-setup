# ai-setup

Готовый набор принципов, скиллов и MCP-конфигураций для AI-assisted разработки.
Устанавливается поверх любого существующего проекта за один шаг.

## Что входит

- **Принципы разработки** — workflow, git, безопасность, качество кода, дизайн-система, SEO
- **Скиллы** — debugging, planning, code review, brainstorming, web-page-architect и другие
- **MCP-серверы** — firecrawl, github, context7, playwright, postgres

Стек определяется автоматически по файлам проекта. Существующие файлы инструкций не перезаписываются.

## Установка

Дай своему AI-агенту такую команду:

```
Прочитай файл BOOTSTRAP.md из репозитория https://github.com/[username]/ai-setup
и выполни установку в текущий проект.
```

Агент:
1. Определит твой AI-инструмент (Claude, Cursor, Gemini и др.)
2. Определит стек проекта по файлам
3. Сгенерирует файл инструкций под твой стек
4. Установит скиллы в `.agents/skills/`
5. Настроит MCP-серверы
6. Создаст структуру памяти в `.agents/memory/`

## Обновление

Чтобы обновить до последней версии принципов и скиллов — запусти ту же команду повторно.
Кастомные части файлов инструкций не затрагиваются.

## Структура репозитория

```
ai-setup/
├── BOOTSTRAP.md          — инструкция для агента
├── core/
│   └── principles.md     — универсальные принципы
├── skills/               — скиллы для агента
└── mcp/
    └── core.json         — список MCP-серверов (без ключей)
```

## Опциональный скилл для продуктовой документации

Для создания PRD, User Stories, эпиков и задач:

```bash
npx @smithery/cli@latest skill add diegoeis/product-spec-kit
```
